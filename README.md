# 咪啪工具集

咪啪工具集是一个功能丰富、界面友好的在线工具集合，旨在为用户提供高效、便捷的文件处理和文本操作解决方案。无论是开发者、运维人员还是普通用户，都能从中找到适合自己的实用工具。

## 🎯 项目目标

- 提供一站式的在线工具服务
- 简化复杂的文件处理流程
- 提高工作效率，减少重复劳动
- 确保数据安全，保护用户隐私
- 支持跨平台、跨设备使用

## ✨ 功能特性

### 1. ZIP文件处理
- 📦 支持上传ZIP文件并快速解析其内容结构
- 📁 直观显示文件和目录的层次关系
- 🔄 支持子压缩包的递归处理，自动识别嵌套ZIP文件
- ✅ 提供文件勾选功能，支持选择性导出
- ⚡ 高效的文件解析算法，处理大型ZIP文件也能保持流畅
- 📤 支持将选中的文件导出为新的ZIP文件

### 2. 文本搜索功能
- 📄 支持多文件同时上传（.txt, .log, .json, .xml, .html, .css, .js等多种格式）
- 🔍 强大的正则表达式搜索，支持复杂匹配模式
- 🔤 提供区分大小写搜索选项
- 🎨 搜索结果高亮显示，直观展示匹配内容
- 📊 显示匹配行号和上下文，方便定位和理解
- 📋 支持搜索结果的复制和导出

### 3. 移除重复行
- 📝 支持直接粘贴文本或上传文件
- 🔁 智能识别并移除文本中的重复行
- ⚙️ 提供多种处理选项，如保留顺序、忽略空格等
- 📊 显示处理前后的行数统计，直观展示处理效果
- 📤 支持处理结果的复制和下载

### 4. JSON格式化
- 📋 支持JSON文本的格式化，使其更易阅读
- 📦 支持JSON压缩，减少文件大小
- ✅ 实时JSON验证，自动检测语法错误
- 🔄 支持JSON与其他格式的转换
- 🎨 提供友好的编辑界面，支持语法高亮
- 👁️ 实时预览功能，所见即所得

### 5. 图片裁切工具
- 🖼️ 支持自定义裁切和圆形裁切模式
- 📷 支持JPG、PNG、GIF和WebP多种图片格式
- 🔍 实时预览裁切效果
- 📏 精确调整尺寸、位置和缩放比例
- 🎨 提供缩放控制滑块和按钮
- 💾 支持裁切结果的下载保存
- 📱 拖拽上传和点击选择图片两种方式

## 🚀 快速开始

### 本地部署

1. **克隆仓库**
```bash
git clone <repository-url>
cd mipa-tools
```

2. **启动本地服务器**

使用Python：
```bash
python3 -m http.server 8000
```

或使用Node.js：
```bash
npx serve
```

或使用PHP：
```bash
php -S localhost:8000
```

3. **访问应用**

在浏览器中输入以下地址：
```
http://localhost:8000
```

## 🔒 隐私与安全

- 🔐 **本地处理**：所有文件处理均在浏览器端完成，不会上传任何文件到服务器
- 📊 **数据隔离**：不同工具之间的数据完全隔离，确保数据安全
- 🕒 **临时存储**：仅在浏览器会话期间临时存储数据，关闭页面后自动清除
- 📝 **无跟踪**：不收集任何用户数据，保护用户隐私

## 📋 使用说明

### 1. ZIP文件处理工具

1. 打开 **ZIP文件处理** 工具页面
2. 点击 "选择文件" 按钮，选择要处理的ZIP文件
3. 等待系统自动解析文件结构
4. 在文件列表中，您可以：
   - 点击文件夹图标展开/折叠目录
   - 勾选需要导出的文件
   - 查看文件大小和类型信息
5. 点击 "导出所选文件" 按钮，将选中的文件导出为新的ZIP文件

### 2. 文本搜索工具

1. 打开 **文本搜索** 工具页面
2. 点击 "选择文本文件" 按钮，上传一个或多个文本文件
3. 在搜索框中输入关键词或正则表达式
4. 根据需要选择 "区分大小写" 选项
5. 点击 "搜索" 按钮开始查找
6. 在搜索结果中，您可以：
   - 查看匹配的文件名、行号和上下文
   - 复制匹配的内容
   - 导出搜索结果

### 3. 移除重复行工具

1. 打开 **移除重复行** 工具页面
2. 选择以下任意一种方式输入文本：
   - 在文本框中直接粘贴文本
   - 点击 "选择文件" 按钮上传文本文件
3. 根据需要选择处理选项：
   - 保留原始顺序
   - 忽略空格差异
   - 忽略空行
4. 点击 "处理" 按钮开始移除重复行
5. 查看处理结果，包括：
   - 处理前后的行数统计
   - 移除的重复行数量
6. 可以选择复制结果或下载处理后的文件

### 4. JSON格式化工具

1. 打开 **JSON格式化** 工具页面
2. 在左侧编辑框中输入或粘贴JSON文本
3. 系统会自动进行以下处理：
   - 格式化JSON，使其更易阅读
   - 验证JSON语法，标记错误位置
4. 您可以：
   - 切换格式化/压缩模式
   - 复制处理后的JSON
   - 下载处理后的文件
   - 清空编辑框，重新输入

### 5. 图片裁切工具

1. 打开 **图片裁切** 工具页面
2. 选择以下任意一种方式上传图片：
   - 点击"选择图片"按钮上传
   - 将图片拖拽到上传区域
3. 选择裁切模式：
   - **自定义裁切**：自由调整裁切区域大小和形状
   - **圆形裁切**：创建圆形裁切区域
4. 调整裁切区域：
   - 拖动裁切框或图片调整位置
   - 使用鼠标拖拽裁切框边缘调整大小
   - 使用缩放控制滑块或按钮调整图片缩放比例
   - 在精确尺寸设置中输入具体的宽度和高度
5. 查看实时预览效果
6. 点击"下载图片"按钮保存裁切结果
